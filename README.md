# Aprendizado de C++

Este repositório contém exemplos práticos e comentados para o aprendizado dos principais conceitos da linguagem C++. Cada arquivo aborda um tema fundamental para quem está começando a programar em C++.

## Estrutura dos Arquivos

- **Variaveis.cpp**: Demonstração de declaração, inicialização e uso de variáveis de diferentes tipos (int, float, double, char, string, size_t, bool).
- **Operadores.cpp**: Exemplos de operadores aritméticos, incremento, divisão inteira, resto, divisão real e potência.
- **Condicionais.cpp**: Uso de estruturas condicionais (`if`, `else if`, `else`) para tomada de decisão com base em médias de notas.
- **Op_Logicos.cpp**: Aplicação de operadores lógicos e condicionais para avaliação de frequência e notas, com exemplos de classificação.
- **Repeticoes.cpp**: Estruturas de repetição (`while`, `do...while`, `for`) para somar números de 1 até N, mostrando diferentes formas de laço.
- **Funcoes.cpp**: Criação e uso de funções, incluindo funções sem retorno e funções que retornam valores.

## Como Usar

1. **Abra os arquivos** no seu editor de preferência (VS Code, Code::Blocks, etc.).
2. **Compile** cada arquivo separadamente usando um compilador C++ (exemplo: `g++ Variaveis.cpp -o variaveis.exe`).
3. **Execute** o programa gerado para ver o resultado e entender o funcionamento do código.

## Adicionando Novos Arquivos de Aprendizado

- Crie um novo arquivo `.cpp` com um nome descritivo do tema abordado.
- Comente o código explicando cada parte, seguindo o padrão dos arquivos existentes.
- Atualize este README.md, adicionando uma breve descrição do novo arquivo na seção "Estrutura dos Arquivos".

## Objetivo

O objetivo deste repositório é servir como material de apoio para quem está iniciando em C++, facilitando o entendimento dos conceitos básicos por meio de exemplos simples e bem comentados.
